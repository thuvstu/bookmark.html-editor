# bookmark.html-editor
google検索時の余計なurlパラメータを消す機能追加
ほかにもユーザー側にまったくもって必要のない余計なURLパラメータがあるサイトは消したいね
そしてEditorを改良したい:
フォルダが一つ以上ないとインポートが正常に開始できない問題を直す
mergerで整理されたフォルダが多すぎて下に行くのが大変。そして下側に必ず表示されるものを見たいなってときがあるのに下に行くのがだるすぎる(左ウインドウのフォルダも、右側のフォルダ群や開いた先のリンク群も)。最下部移動とか、あとページを長くするんじゃなくて中に入れる(overflow と height使ってボックスに高さを指定し、はみ出た分をスクロールで見せる,<iframe>,object,embed)とかでなんとかする。
左ウインドウと開くボタンからフォルダを開けるけど名前変更の当たり判定短くして(名前の部分だけとかにする)フォルダ開ける範囲を追加(フォルダごとの何も書かれていない空白部分)してくれ。(ボタン系はいじらなくてよい)
複数選択ができないのが致命的。
ホールド(ドラッグアンドドロップ)でもフォルダやリンクの位置をかえられるように。
リンク先に飛びにくいのでフォルダ開くのと同じように使いやすくして。
必要に応じてダブルクリックの必要性(一度おして青くなった状態でもう一度押すみたいな、普通のブックマークリーダーにあるようなシステム)をつけてもよいと思います。
